# Personal Discord Bot

A multi-purpose Discord bot designed for personal assistance, built with modularity in mind.

## Features

### Fun & General
*Powered by the **Krypton** template.*
- Includes a variety of general-purpose and entertainment commands.
- Modular structure for easy expansion.

### Stock Market
*Tools for tracking the market.*
- **Real-time Data:** Fetch live stock market data instantly.

### Competitive Programming (CP)
*Complete toolkit for algorithm practice and progress tracking.*
- **Random Problems:** Fetch weighted or completely random problems from Codeforces and AtCoder.
- **Daily Challenges:** Automated daily problem distribution with streak tracking.
- **Account Management:** Register and verify Codeforces and AtCoder handles.
- **Submission Tracking:** Auto-check submissions and update completion status.
- **Leaderboard System:** Track user progress with streak counts and solved problem stats.
- **Profile Display:** View detailed Codeforces profile information and statistics.
